Ni WooCommerce Sales Report by user role provide the option to link the sales order with your sales agent or sales person.

== Description ==

<h3>Ni WooCommerce Sales Report By User Role Description</h3>
<p>
Ni WooCommerce Sales Report by User Role provide the flexibility to check the sales report base on the sales person or sales agent. This plugin help to find the sales person or sales agent contribution on sales.
</p>
<p>
Ni WooCommerce Sales Report by User Role provide the sales analysis report base on the user role. This report provide the ability to assign the sales agent with order and provide the sales analysis report base on the selected sales agent.
</p>
<p>
This report also provide the option to filter the report base on the sales agent  sales and order period like today sales, yesterday sales and this month sales.
</p>
<p>
 We have provide the flexibility option to select your role for sales agent or select <strong>Ni Sales Agent</strong> for new role. Create the new sales agent base on selected role and assign that sales agent in order.
</p>

== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate plugin from admin panel.
3. Find the menu User Role Report. 
4. Go to setting and select the existing  user  role or select the <strong> Ni Sales Agent</strong> for new user role.
5. Go to user add new user and select the setting role or  <strong>Ni Sales Agent</strong> role.
6. Edit the order and find select sales agnet dropdown and select user.
7. Go to sales order list and enjoy the plugin.
